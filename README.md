# Minigames-hub

A web-site about  3 minigames(by now) made by fun and also self-learning using HTML, CSS and JS

## Features

- **Rock, Paper & Scissors**, **Quiz Game**, and **Flip a Coin** minigames  
- Consistent and visually appealing color scheme  
- Fully **responsive** and mobile-friendly design  
- Simple and intuitive UI for quick gameplay

## Future Updates

- Add more minigames (ideas coming soon!)  
- Improve animations and game feedback  
- Add scoreboards or leaderboards for each game
